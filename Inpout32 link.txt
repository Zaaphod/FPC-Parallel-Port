Instructions:
Download InpOut32 "Binaries only - x86 & x64 DLLs and libs."
from http://www.highrez.co.uk/downloads/inpout32/
Extract Acrchive
run InstallDriver.exe
go to InpOutBinaries_1501\Win32 folder or InpOutBinaries_1501\Win64
Put inpout32.dll in the same folder as your Pascal files if you will be compiling your program for Win32
Put inpoutx64.dll in the same folder as your Pascal files if you will be compiling your program for Win64
Compile
Run
*note parallel port is in hex, but demo program doesn't convert it for you.
 enter $378 to indicate hex or use decimal 888 for port 0x378
